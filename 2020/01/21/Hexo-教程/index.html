Hexo 教程一---本地搭建

字数统计: 409字   |   阅读时长≈ 1分钟

¶前言

作为一个编程爱好者，一直希望拥有一个自己的博客网站，但是由于时间的关系，一直没有能够动手搭建。在春节期间，闲来无事，开始着手搭建自己的博客。为什么选择hexo，主要是因为Hexo是一个快速、简洁且高效的静态博客框架，适合小白。下面记录了我搭建hexo博客的一些主要步骤。因为我自己也是一个小白，所以搭建流程会更可能的详细。

¶1 环境搭建

  • Node.js
  • npm
  • git

Node.js中已经带有了npm。
Node.js和git的安装请自行百度

完成安装之后，win+r 输入cmd 并打开，依次输入node -v、npm -v 和 git --version 并回车，出现下图版本号则说明安装成功。

¶2 本地安装 hexo 博客程序

¶2.1安装Hexo

使用 npm一键安装 Hexo 博客程序

npm install -g hexo-cli

安装时间可能会有点久，由于我已经安装过了，再次运行该命令，会出现如下图所示的界面

¶2.2本地初始化

在本地新建一个文件夹 Hexo_Blog (文件夹名称随意)。进入该文件夹，右击 -> Git Bash Here。如下图所示

初始化并安装所需软件：

hexo init #初始化

hexo install #安装组件

安装过程

安装之后，新建文件夹中的文件目录结构

完成之后依次输入以下命令，启动本地服务器进行预览

hexo g

hexo s

访问 http://localhost:4000, 出现 Hexo 默认页面，本地博客安装成功！
