¶前言

Hexo 为什么流行，不仅因为其安装部署简单，并且它提供了大量主题样式，开发者可以选择自己喜欢的样式，并且更改样式非常简单。本文将以 Ayer 为例（本文较为喜欢这种风格），详细介绍步骤以及 Ayer 的一些特性。

¶Ayer 的特性

Ayer 是一个干净优雅的 Hexo 的主题，自带响应式，记载速度快，具有很多特性。（本博客就是应用此主题）

  1. 干净优雅，内容美观易读
  2. 响应式设计，桌面版、平版、手机等设备均能有较好的展现
  3. 时间轴式的归档页
  4. 侧边栏可以点击显示与隐藏
  5. 目录在文章页悬浮，跳转方便
  6. 集成了 Valine 和 Gittalk 评论模块

参考沈宇的博客

¶安装步骤

首先打开博客目录，输入以下命令 git clone https://github.com/Shen-Yu/hexo-theme-ayer.git themes/ayer

git clone https://github.com/Shen-Yu/hexo-theme-ayer.git themes/ayer

此时可以在博客主目录 /themes 下看到多了一个文件夹，名称是 Ayer。

打开博客主目录下的 _config.yml 文件，把 hexo 默认的主题 landscape 改成 ayer，如下图所示

运行 hexo g -d，然后再运行 hexo s，就会看到效果。

至此，更换主题就完成了。这个主题的其他的一些特色会在之后的教程中介绍，比如为你的博客添加评论系统、博客目录管理之类的。
